# Página de Receita

Projeto simples de uma página estática com uma receita de **cupcake de café com chantilly**, desenvolvido com HTML e CSS.

## 📄 Sobre

Esta página apresenta:

- título e descrição da receita;
- tempo, rendimento e dificuldade;
- lista de ingredientes;
- modo de preparo;
- imagem principal da receita.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3

## 📁 Estrutura do projeto

```bash
.
├── index.html
├── style.css
└── assets/
```

## ▶️ Como executar

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto no VS Code.
3. Abra o arquivo `index.html` no navegador.

> Dica: se usar a extensão **Live Server**, clique em “Go Live” para visualizar a página automaticamente.

## 🎯 Objetivo

Praticar estruturação de conteúdo com HTML e estilização com CSS em uma landing page simples de receita.
